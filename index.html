Built for Security

Advanced isolation patterns to keep your data safe.

Hardened Security Defaults

Every container runs as a non-root user with dropped Linux capabilities. We follow the principle of least privilege, ensuring AI agents only have access to what they truly need.

No Root Access Seccomp Profiles Resource Limits

Granular SSH Selection

Explicitly select which SSH keys and host configurations are exposed to the sandbox. No more global credential exposure.

Workspace Isolation

Only whitelisted directories are mounted into the container. Keeping your system files completely invisible to the agent.

Instant Start

Pull pre-built images from the GitLab registry. Zero build time, just local security for global AI power.

Continuous Integration

Every image is automatically built, tested, and scanned for vulnerabilities in our GitLab CI pipeline. Security is baked in from commit one.
